as the backend represented by [[ref:acb3d22f-5b93-4e95-b73c-f1715523877e][url]] satisfies [[ref:e58ffb29-988b-4423-a5d7-2d5c41b58ff1][this contract]].

** frontend#fetch
:PROPERTIES:
:ID:       c6dfad7d-72df-4788-ac6f-97c297f6df94
:END:

Premises
- The [[ref:6612d34b-e5e7-4d42-b1bf-d73bb1ed9906][environment]] is started
- repo : A URL where to fetch the code of the front end from ; /e.g./ "git@github.com:phf-1/my-lovely-tasks.git"
- frontend#fetch repo

Algorithm
- cond
  - [[ref:834f11b4-dcdf-46c2-8472-15884c46ed98][frontend#dir]] does not exist ↦ repo cloned into it
  - _ ↦ cond
    - remote = repo ↦ git#pull ■
    - remove ≠ repo ↦ match user#confirmation
      - y ↦ delete(frontend#dir) frontend#fetch(repo) fetch(npm dependencies) ■
      - _ ↦ ■

Conclusion
- operation aborted or repo has been cloned under frontend#dir and npm dependencies
  have been fetched
