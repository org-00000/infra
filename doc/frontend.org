** frontend#fetch
:PROPERTIES:
:ID:       c6dfad7d-72df-4788-ac6f-97c297f6df94
:END:

Premises
- The [[ref:6612d34b-e5e7-4d42-b1bf-d73bb1ed9906][environment]] is started
- repo : A URL where to fetch the code of the front end from ; /e.g./ "git@github.com:phf-1/my-lovely-tasks.git"
- frontend#fetch repo

Conclusion
- The last version of the frontend has been fetched from repo into [[ref:834f11b4-dcdf-46c2-8472-15884c46ed98][frontend#dir]] as
  well as its dependencies.

** frontend#dist-mk
:PROPERTIES:
:ID:       5357e215-36ce-4592-839d-7032b21b5b62
:END:

Premises
- The last version of the frontend has been fetched.
- backend-url : The URL of the [[ref:43703ca8-74e9-454e-95c6-566ca4f8bbc7][backend-url]] ; /e.g./ "http://localhost:4000/api"
- frontend#dist-mk backend-url

Conclusion
- A [[https://vite.dev/guide/build][distribution]] has been built and stored into [[ref:985496c0-8e69-4545-9dc2-e9c830e02581][frontend#dist]] such that requests are
  directed towards backend-url.
