** owner_permission
:PROPERTIES:
:ID:       5aa7695b-3054-43d5-a1a9-aa3d3973b14d
:END:

Premises
- owner : User ; /e.g./ =alice=
- path : SelectedFile ; /e.g./ =$INFRA/dir=
- owner is the inode owner of path
- permission :≡ owner_permission(owner path)

Conclusion
- permission : String that represents path's owner's permission bits ; /e.g./ "rwx"

** transfer_permission_on_change
:PROPERTIES:

:END:

Conclusion
- inotifywait is started before transfer_all_permissions
- Eventually, for any =f= : SelectedFile that has been created or whose

  is expected to restart the watcher after a =mv= if a full sync is needed.
