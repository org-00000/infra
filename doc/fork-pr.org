Fork
1. Fork the repository: https://github.com/org-00000/infra
2. git remote add upstream git@github.com:org-00000/infra.git

PR
1. Define or choose a [[ref:3427d8a4-4fc0-4d98-a8f7-59ac1399a9dc][task]] in the [[ref:90abac79-d226-4de2-9174-9df6c48c25c0][backlog]].
2. Make sure that the task is well understood, talk with the team: update the task
   accordingly.
3. Assign yourself to the task.
4. Add/update/delete relevant [[ref:1f6eeb40-6e6b-4e19-9d5d-23af3858023d][specifications]] and implementations.
5. Send a [[ref:5deb7507-29b1-44da-9286-a81ced5f594c][PR]].
6. Wait for the PR to be merged. You may work on an other local branch in the meantime.
7. Synchronize with upstream: [[ref:61e5914d-eb8d-46cf-8b7c-31b68079c83e][,synchronize-with-upstream]]

- [ ] The code compiles cleanly.
